HTML <s> tag is used to represent the range of content which is no longer accurate or relevant in some ways. The text written between <s>(start) and </s> (end) tag, renders as a strike a line through the text.

Syntax:

Code Example

<s> Write your content here </s>

Following are some specifications about HTML <s> tag.

DisplayInline
Start tag/End tagBoth Start and End tag.
UsageFormatting

Attribute:

The <s> tag does not contain any specific attribute.

The <s> tag supports both the global and event attributes in HTML.
