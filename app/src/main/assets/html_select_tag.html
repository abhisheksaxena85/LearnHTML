HTML <select> tag is used to create a drop down list with multiple options. The <options> element is nested within <select> tag for defining options in a list.

The <optgroup> element can be used for grouping related options in a list.

If you want to sent data to server then use <select> tag within <form> element.

Syntax:

Code Example

<select>
<option> </option>
</select>
