The HTML <center> is a block level element which contains both block level and inline contents within it. The content written between the <center> elements will be displayed will be displayed at the middle of the page.

The <center> tag has been derecated in HTML4 and obsolete in HTML5.

Syntax:

Code Example

<center>Add content here....</center>

Output:

Add content here....

Following are some specifications about the <center> tag.

DisplayInline
Start tag/End tag.Both start and End tag.
UsageTextual

Attribute

HTML <center> tag does not contain any specific attribute in HTML but it supports the Global Attribute (till <center> tag is not removed completely).
