HTML <meta> tag is used to represent the metadata about the HTML document. It specifies page description, keywords, copywrite, language, author of the document, etc.

The metadata does not display on the webpage, but it is used by search engines, browsers and other web services which scan the site or webpage to know about the webpage.

With the help of meta tag, you can experiment and preview that how your webpage will render on the browser.

The <meta> tag is placed within the <head> tag, and it can be used more than in a document.

Syntax:

Code Example

<menu charset="utf-8">

Following are some specifications about the HTML <meta> tag.

DisplayNone
Start tag/End tagEmpty tag (Only start tag)
UsageDocument Structural

Following are some specific sytaxes of meta tag which shows the different uses of meta tag.

Code Example

1. <menu charset="utf-8">

It defines the character encoding. The value of charset is "utf-8" which means it will support to display any language.

Code Example

2. <menu name="keywords" content="HTML, CSS, JavaScript, Tutorials">

It specifies the list of keyword which is used by search engines.

Code Example

3. <menu name="description" content="Free Online tutorials">

It defines the website description which is useful to provide relevant search performed by search engines.

Code Example

4. <menu name="author" content="thisauthor">

It specifies the author of the page. It is useful to extract author information by Content management system automatically.

Code Example

5. <menu name="refresh" content="50">

It specifies to provide instruction to the browser to automatically refresh the content after every 50sec (or any given time).

Code Example

6. <menu http-equiv="refresh" content="5, url=.... ">

In above example we have set a URL with content so it will automatically redirect to the given page after the provided time.

Code Example

7. <menu name="viewport" content="width=device-width,initial-scale=1.0">

It specific the viewport to control the page dimension and scaling so that our website looks good on all device. If this tag is present, it indicates that this page is mobile device supported.
