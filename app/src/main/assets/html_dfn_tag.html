HTML <dfn> tag is also called as HTML definition tag. It is used to represent the term which is defined within context of definition phrase or sentence in an HTML document. The defining instance term usually the first term in a document.

If a term contained within the <dfn> element then browser understands that nearby text is the definition of the term.

Syntax:

Code Example

<dfn> Content..... </dfn>

Following are some specifications about the HTML <dfn> tag.

DisplayInline
Start tag/End tagBoth start and End tag
UsageTextual

Attribute

AttributeValueDescription
titleIt specifies the definition of the term and it shows when mouse hover on it.

HTML <dfn> tag supports both Global and Event attributes in HTML.
