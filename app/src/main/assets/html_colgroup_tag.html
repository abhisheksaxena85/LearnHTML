HTML <colgroup> tag specifies group of column within an HTML table. It is used a parent container of one or more <col> elements to apply different properties in an HTML table.

Syntax:

Code Example

<colgroup>.....</colgroup>


Following are some specifications about HTML <colgroup> tag.

DisplayNone
Start tag/End tagStart tag and End Tag
UsageHTML Tables

Attributes

AttributeValueDescription
alignleft, center, right, justify, charIt specifies the alignment of the column content.
charcharacterIt specifies the alignment of the content in a column group to the character.
charoffnumberIt sets the number of characters to offset the column data from the alignment character specified by the char attribute.
spannumberIt specifies the number of columns a colgroup should span.
valigntop. middle, bottom, baselineIt specifies the vertical alignment of the column group
widthpixel, %, relative_lengthIt specifies the width of the group of column.

HTML <colgroup> tag supports both Global and Event attributes in HTML.
