HTML <hr> tag is used to specify a paragraph-level thematic break in HTML document. It is used when abruptly change your topic in your HTML document. It draw a horizontal line between them. It also called a Horizontal Rule in HTML.

Code Example

<h4>HTML is a language for describing web pages. </h4>
<hr/>
<p>This is the content of web page.</p>

Output:

HTML is a language for describing web pages.

This is the content of web page.


hr tag in HTML 4.01 and HTML5?

In HTML 4.01, the <hr> tag represents a horizontal rule while in HTML5, it defines a thematic break. CSS is used in HTML5 instead of layout attributes.

hr tag in HTML and XHTML

In HTML <hr> tag need not to be closed whereas <hr> must be properly closed in XHTML.
