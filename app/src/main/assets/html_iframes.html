HTML iframe is used to display a nested webpage (a webpage within a webpage). The HTML <iframe> tag defines an inline frame.

An HTML iframe embeds another document within the current HTML document in the rectangular region.

The webpage content and iframe contents can interact with each other using JavaScript.

Iframe Syntax

An HTML iframe is defined with the <iframe> tag:

Code Example

<iframe src="URL"> </iframe>

Here, "src" attribute specifies the web address (URL) of the inline frame page.

Set Width and Height of iframe

You can set the width and height of iframe by using "width" attributes. By default, the attribute's values are specified in pixels but you can also set them in percent. i.e. 50%, 60%, etc.

Example : (Pixels)

Code Example

<iframe src="https:/ /www.google.com/" width="100" height="100" > </iframe>


Example : (Percentage)

Code Example

<iframe src="https:/ /www.google.com/" width="40%" height="40%" > </iframe>

You can also use CSS to set the height and width of the iframe.

Example:

Code Example

<iframe src="https:/ /www.google.com/" style="width:200px; height:100px;" > </iframe>

Remove the border of iframe

By default, an iframe contains a border around it. You can remove the border by using <style> attribute and CSS border property.

Example:

Code Example

<iframe src="https:/ /www.google.com/" style="border:none;" > </iframe>

You can also change the size, color, style of the iframe's border.

Example:

Code Example

<iframe src="https:/ /www.google.com/" style="border:2px solid black;"> </iframe>

Iframe Target for a link

You can set a target frame for a link by using iframe. Your specified target attribute of the link must refer to the name attribute of the iframe.

Example:

Code Example

<iframe src="action.html" style="width:200px; name="iframe" height:100px;" > </iframe>
<p> <a href="https:/ /www.google.com/lab" target="iframe"> google</a>

Output:

google

Enbed YouTube video using iframe

You can also add a YouTube video on your webpage using the <iframe> tag. The attached video will be played at your webpage and you can also set height, width, autoplay, and many more properties for the video.

Following are some steps to add YouTube video on your webpage:

  • Goto YouTube video which you want to embed.
  • Click on SHARE → under the video.
  • Click on Embed <> option.
  • Copy HTML code.
  • Paste the code in your HTML file.
  • Change height, width, and other properties (as per requirement).

Example:

Code Example

<iframe width="560" height="315" src="https:/ /www.youtube.com/ embed/o5tccfuC4pI?si=- fAgN3qjanmMzHZn" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen></iframe>

Output:
