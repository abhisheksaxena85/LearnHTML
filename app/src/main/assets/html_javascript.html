A script is a small program which is used with HTML to make web pages more attractive, dynamic and interactive, such as an alert popup window on mouse click. Currently, the most popular scripting language is JavaScript used for websites.

Code Example

<!DOCTYPE html>
<html>
<body>
<h1>JavaScript Date and Time </h1>
<button type="button" onclick="document. getElementById('demo') .innerHTML = Date()"> Click on Button </button>
<p id="demo"> </p>
</body>
</html>

Output:

JavaScript Date and Time

HTML <script> Tag

The HTML <script> tag is used to specify a client-side script. It may be an internal or external JavaScript which contains scripting statements, hence we can place <script> tag within <body> or <body> section.

It is mainly used to manipulate images, form validation and change content dynamically. JavaScript uses document.getElementById() method to select an HTML element.

Code Example

<!DOCTYPE html>
<html>
<body>
<h1>Use JavaScript to change text. </h1>
<button type="button" onclick="document. getElementById('demo_text') .innerHTML = 'Hello'"> Click on Button </button>
<p id="demo_text"> </p>
</body>
</html>

Output:

Use JavaScript to change text.

HTML events with JavaScript

An event is something which user does, or browser does such as mouse click or page loading are examples of events, and JavaScript comes in the role if we want something to happen on these events.

HTML provides event handler attributes which work with JavaScript code and can perform some action on an event.

Syntax:

Code Example

<element event="JS code" >

Example

Code Example

<input type="button" value="click" onclick="alert('Hello World')">

Output:


HTML can have following events such as:

  • Form Events: reset, submit, etc.
  • Select Events: text field, text area, etc.
  • Mouse Events: select, mouseup, mousemove, mousedown, click, dbclick, etc.

Following are the list for Window event attribute:

Event NameHandler Name Occurs when
onBlurblurWhen form input loses focus.
onClickclickWhen the use clicks on a form element or a link.
onSubmitsubmitWhen user submits a form to the server.
onLoadloadWhen page loads in a browser.
onFocusfocusWhen user focuses on an input field.
onSelectselectWhen user selects the form input field.

Let's see what JavaScript can do:

1) JavaScript can change HTML content.

Example

Code Example

<!DOCTYPE html>
<html>
<body>
<p>JavaScript can change the content of an HTML element.</p>
<button type="button" onclick="myFunction()">Click Here</button>
<p id="text_change"></p>
<script>
function myFunction(){ document.getElementById ('text_change') .innerHTML = 'Hello World';
}
</script>
</body>
</html>

Output:

JavaScript can change the content of an HTML element.

JavaScript can change HTML style


Example

Code Example

<!DOCTYPE html>
<html>
<body>
<p id="style_change" >JavaScript can change the content of an HTML element.</p>
<button type="button" onclick="myFunction()">Click Here</button>
<script>
function myFunction(){ document.getElementById ('style_change').style. fontSize = "32px";
document.getElementById ('style_change').style. color = "red";
document.getElementById ('style_change').style. transition = '1s ease';
}
</script>
</body>
</html>

Output:

JavaScript can change the content of an HTML element.


JavaScript can change HTML attribute

With the help of JavaScript we can change the HTML attributes as per our requirement.


Use External Script

Suppose, you have various HTML files which should have same script, then we can put our JavaScript code in separate file and can call in HTML file. Save JavaScript external files using .js extension.

Syntax:

Code Example

HTML <noscript> Tag

HTML <noscript> tag is used to write disabled script in the browser. The text written within <noscript></noscript> tag is not displayed on the browser.

Syntax:

Code Example

<noscript>This text is not visible in the browser.</noscript>
