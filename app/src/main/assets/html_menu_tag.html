HTML <menu> tag specifies a list or menu of commands that a user can perform or activate. It is used for creating context menu as well as lists menu.

A <menu> element can contain one or more <li> or <menuitems> elements within it.

Syntax:

Code Example

<menu> ..... </menu>

Following are some specifications about the HTML <menu> tag

DisplayInline
Start tag/End tagEmpty tag (Only start tag)
UsageSemantic and textual

Attribute:

AttributeValueDescription
labeltextIt specifies the label for the menu.
typeIt specifies the type of the menu in an HTML document.

The <menu> tag supports both the global and event attributes in HTML.
