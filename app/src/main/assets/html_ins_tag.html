HTML <ins> tag is used to represent the newly added text in an HTML document. It usually renders with an underline through the text, but it can be changed using CSS property.

The <ins> tag is more useful when it is used with the <del> element as it represents the part of the text that has been deleted and <ins> represent the part of the text that has been inserted to the document.

Syntax:

Code Example

<ins>Insert the text...</ins>

Following are some specifications about the HTML <ins> tag.

DisplayInline
Start tag/End tagBoth start and end tag
UsageSemantic

Attributes

AttributeValueDescription
citeURLIt specifies the URL of the resource which explains the reason for the change.
datetimeYYYY:MM:DD HH:MM:SSIt specifies the date and time of change.

HTML <ins> tag supports all the global and event attributes in HTML.
