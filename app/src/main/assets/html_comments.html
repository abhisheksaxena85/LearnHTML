Comments are some text or code written in your code to give an explanation about the code, and not visible to the user. Comments which are used for HTML file are known as HTML comments. Anything written between these tags will be ignored between these will be ignored by the browser, so comments will not be visible on the webpage.

Comments of any code make code easy to understand and increase readabilty of code.

Comments are also part of the code, which gives an explanation of the code.


How to add comment in HTML

You can add comments in your HTML file using <! -- .... --> tag. So if you will write anything between these comment tag that will be treated as comment and browser will not read it.

Syntax:

Code Example

<!-- Write commented text here -->

Such as:

Code Example

<!-- <p>This is some Text.</p>
        <p>This is second text.</p> -->


Multiline Comment

In HTML code, we can also comments multiple lines at a time. In multiline comment we can use any description about code or multiple line code to debug, etc.

Example:

Code Example

<!-- Your code is commented.
Write description of code.
It will not display at webpage.
-->
