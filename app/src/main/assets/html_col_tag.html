HTML <col> tag is used to specify common column properties for each column within <colgroup> element in an HTML table.

The <col> tag is useful when it requires to apply common style rules for each column cell rather than applying individually.

Syntax:

Code Example

<col span=' ' style=' ' >

Following are some specifications about the HTML <col> tag.

DisplayInline
Start tag/End tagStart tag(Empty tag)
UsageHTML tables

Attribute:

Tag-specific attribute element should contain. The default value of span attribute is 1.

AttributeValueDescription
alignleft, center, right, justify charIt specifies the horizontal alignment of each column cell.
charcharacterIt specifies the alignment of content based on characters in a column. It will be ignored if align is not set to char.
charoffnumberIt sets the number of characters to offset the column data from the alignment character specified by the char attribute.
spannumberIt specifies the number of the columns.
valigntop, middle, bottom, baselineThe specifies the vertical alignment of the columns.
width%, pixel, relative_lengthIt specifies the width of the column.

HTML <col> tag supports all Global and Event attributes.
