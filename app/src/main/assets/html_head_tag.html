The HTML <head< element is used as a container for metadata (data about data). It is used between <html> tag and <body> tag.

The head of an HTML document is a part whose content is not displayed in the browser on page loading. It just contains metadata about the HTML document.

An HTML head can contain lots of metadata information or can have very less or no information, it depends on our requirement. But head part has a crucial role an HTML document while creating a website.

Metadata defines the document title, character set, style, links, scripts and other meta information.

Following is a list of tags used in metadata:

  • <title>
  • <style>
  • <meta>
  • <link>
  • <script>
  • <base>

HTML <title> Element

The HTML <title> element is used to define the title of document. It is used in all HTML/XHTML documents. The <title> element must be placed between <head> element, and one document can only have one title element.

What does <title> element do?

  1. It defines a title in the browser tab.
  2. It provides a title for the page when it is added to favorites.
  3. It displays a title for the page in search engine results.

Example:

Code Example

<!DOCTYPE html>
<html>
<head>
<title> This is the page title </title>
</head>
<body>
</body>
</html>

HTML <style> Element

The HTML <style> element is used to style the HTML page. The <style> element can have CSS properties for that HTML page only. If we want to apply CSS for multiple pages then we should use separate CSS file.

Example:

Code Example

<!DOCTYPE html>
<html>
<head>
<title> This is the page title </title>
<style>
h4{
color:red;
}
</style>
</head>
<body>
<h4>This is heading<h4>
</body>
</html>

Output:

This is heading

HTML <link> Element

The HTML <link> element is used to link an external style sheet to your webpage. The <link> element contains main two attributes which are "rel" and "href". The rel attribute indicates that it is a stylesheet, and href gives the path to that external file.

Syntax:

Code Example

<link rel="stylesheet" href="style.css">

HTML <meta> Element

The HTML <meta> element is used to specify the character set, page description, keyword, authors and other metadata on the webpage.

Metadata is mainly used by browsers, search engines, and other web services to rank better.

Let's see how to use metadata:

To define a character set:

Code Example

<meta charset="UTF-8" >

The charset attribute specifies the character encoding. In this example we have set it to "UTF-8" which means it can handle to display any language.

Example:

Code Example

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title> This is the page title </title>
</head>
<body>
<h4>This is heading<h4>
</body>
</html>


To define a description of your webpage:


Code Example

<meta name="description" content="Free Web tutorial">

If you give a meta description then it will be useful for the relevant search to perform by search engines.

To define keywords for search engines:


Code Example

<meta name="description" content="HTML, CSS, XML, JavaScript" >

The keyword value is also used to provide keywords for a search engine, but it may ignore by browser due to spammers.

To define author of webpage:


Code Example

<meta name="author" content="John Doe" >

The author value specifies the name of the person who wrote the page content, and it is useful to automatically extract author information by some content management systems.

To refresh document every 30 second:


Code Example

<meta http-equiv="refresh" content="30" >

Meta refresh is used to provide instructions to the browser to automatically refresh the page after the given time interval. As in above example it will automatically refresh after every 30 second.

Code Example

<meta http-equiv="refresh" content="10; url=https://www.google.com/html-head" >

If you add and URL with content value, then it will redirect to that page after the time limit will over.

Use <meta> tag to set the Viewport

This method is introduced in HTML5 to take control over the viewport bt using <meta> tag.

Viewport is the user's visible area of a webpage. It changes from device to device and appears smaller on mobile phones than computer screen.

Syntax:

Code Example

<meta name="viewport" content="width=device-width, initial-scale=1.0" >

Here the <meta> viewport element specifies how to control the page's dimensions and scaling.

The width=device-width is used to set the width of the page, to follow the screen-width of device (which will vary depending on the device).

The initial-scale=1.0 is used to set the initial zoom level when the page first loaded by the browser.
