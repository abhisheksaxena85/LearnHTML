HTML <dir> tag was used as a container for the list of files/folders or content of the directory. The <dir> element is used with <li> tags, and list of directories renders in bullets by default.

Syntax:

Code Example

<dir> Directory list..... </dir>

Following are some specifications about the HTML <dir> tag

DisplayBlock
Start tag/End tagBoth start and End tag
UsageHTML lists

Attribute

AttributeValueDescription
compactcompactIt specifies that list should display smaller than normal.

HTML <dir> tag supports all the Global and Event attributes.
