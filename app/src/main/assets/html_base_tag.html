HTML <base> tag is used to specify a base URI or URL to use for all relative links contained within an HTML document.

Only one <base> element can be specified within a document, and it must be placed with <head> element. We can also specify how other links should open using the target attribute.

Syntax:

Code Example

<base href="https://www. google.com">

In HTML <base> element does not contain closing tag but in XHTML end tag but in XHTML end tag </base> is required.

Following are some specifications about the <base> tag.

DisplayNone
Start tag/End tagOnly start-tag
UsageAnchor and links

Example:

Open in same window (by default)

Code Example

<!DOCTYPE html>
<html>
<head>
<title> </title>
<base href="https://www. google.com">
</head>
<body>
<a href="/pictures">Pictures</a>
<a href="/email">Email</a>
<a href="/calender">Calender</a>
</body>
</html>

Output:

Pictures Email Calender

In the above example we have used base URL = "https:// www.google.com" which all other relative links will treat as starting URL. Here (/) defines the root URL of the current document.

Attribute

Tag specific attributes:

AttributeValueDefinition

target
_blankOpen the relative link in the new window.

The <base> tag supports both global and event attributes in HTML.
