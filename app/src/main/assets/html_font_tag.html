HTML <font> tag is used to define the font style for the text contained within it. It defines the font size, color, and face or the text in an HTML document.

Syntax:

Code Example

<font size="" color="" face="" > Content here.... </font>

Following are some specifications about the HTML <font> tag

DisplayInline
Start tag/End tagBoth start and end tag
UsageFont style

Attributes

AttributeValueDescription
colorrgb(X,X,X), #XXXXX, color_nameIt specifies the color of the content.
facefont_familyIt specifies the typeface of the content.
sizenumberIt specifies the size of the content.
