HTML <small> tag makes text font by one size smaller than document's base font size (Such as large to mediumm, medium to small, etc.)

In HTML5, <small> tag is used for identifying secondary importance such as copyright, side comments, and legal notices.

Syntax:

Code Example

<small> Write Your content here.. </small>

Following are some specifications about the HTML <small> tag

DisplayInline
Start tag/End tagBoth Start and End tag

Attribute

The <small> tag does not contain any specific attribute.

The <small> tag supports the global and event attributes in HTML.
