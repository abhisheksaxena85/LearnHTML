HTML <frameset> tag is used to contain the group of frames which can be controlled and styled as a unit. The <frameset> element also specifies the number of rows and columns in the frameset, and how much spce they will occupy in a frame.

Syntax:

Code Example

<frameset cols=" " >.... </frameset>

Following are some specifications about the HTML <frameset> tag.

DisplayBlock
Start tag/End tagBoth start and End tag
UsageFrames

Attribute
