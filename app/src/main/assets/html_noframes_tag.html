HTML <noframes> tag is used to contain an alternative text to display if the browser does not support the <frame> content. It will only work, if the browser does not support the frame else, it will be ignored by the browser.

The <noframes> tag should be used within <frameset> element.

Syntax:

Code Example

<noframes> ..... </noframes>

Following are some specifications about the HTML <noframes> tag.

DisplayBlock
Start tag/End tagBoth start and End tag.
UsageWith Frames

Attribute

The <noframes> tag does not contain any specific attribute in HTML.

The <noframes> tag only supports the global attributes in HTML.
