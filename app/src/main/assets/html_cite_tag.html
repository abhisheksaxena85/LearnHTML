HTML <cite> tag specifies a citation, it provides reference or title to a creative work, quoted content, books, websites, a resech paper, a blog-spot, painting, etc.

The main pupose of adding <cite> element is to provide or identify the source of the quote or any content.

The content written between <cite> tag renders in the italic form on the browser, and it can be overridden using CSS in your HTML document.

Syntax:

Code Example

<cite>Cited content</cite>


Following are some specifications about the <cite> tag.

HTML <cite> tag does not contain any specific attribute.

HTML <cite> supports both all Global and Event attributes in HTML.
