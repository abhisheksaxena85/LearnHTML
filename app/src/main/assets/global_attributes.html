HTML global attributes are those attributes which are common for all HTML elements. The global attributes are supported by both standard and non-standard elements.

The global attributes can be used with all elements, although it may not have any effect on some elements.

Following is the complete list of global attributes with their description:

AttributesDescription
accesskeyIt is used to generate keyboard shortcuts for current elements.
classIt is used to provide the class name for the current element. It is mainly used with the stylesheet.
contenteditableIt determines whether the content within an element is editable or not.
contextmenuIt defines the id for the <menu> element which is used as a context menu for an element.
data-It is used to store element-specific private data which can be accessed by javascript.
dirIt specifies the direction of the content inside the current element.
draggableIt specifies whether the content within an element is movable or not using Drag and Drop API.
dropzoneIt specifies the action is taken on the dragged element when it is dropped, such as whether it is copied, moved or linked.
hiddenIt is used to hide the element from view.
idIt specifies a unique id for the element. It can be used with CSS and javascript.
langIt specifies the primary language for the content of an element.
styleIt is used to apply inline CSS to the current element.
spellcheckIt specifies whether the content should be checked for spelling errors or not.
tabindexIt determines the tabbing order of an element.
titleIt is used to provide the title, name, or some extra information about the element.
translateIt specifies whether the content of the element should be translated when the page is localized or not.
