HTML <i> tag is used to represent a part of text in a different voice from the surrounding text. The content within <i> tag usually renders in italic type on the browser. It can be useful to represent some technical terms, phrase, fictional charcter thought, etc.

Syntax:

Code Example

<i>Write your content here.. </i>

Following are some specifications about the HTML <i> tag.

DisplayInline
Start tag/End tagBoth start and end tag
UsageTextual

Attribute

HTML <i> tag does not contain any specific attribute.

HTML <i> tag supports both all global and event attributes in HTML.
